# IG Archive Viewer

A fully local Instagram archive analyzer.

This tool allows you to explore your Instagram export directly in your browser without uploading any data to external servers.

## Features

- View followers and following lists
- Detect non-mutual follows
- Explore liked posts
- Browse comments
- Inspect archive files
- Search through data instantly
- Fully offline processing
- No account login required

## Privacy

All processing happens locally in your browser.

No uploads.
No tracking.
No analytics.
No server.

Your archive never leaves your device.

## Technologies

- HTML
- CSS
- Vanilla JavaScript
- JSZip

## Usage

1. Download your Instagram archive from Instagram.
2. Open `index.html`
3. Drag and drop the ZIP archive.
4. Explore your archive locally.
